# To build and test
I've added some batch files for windows setup.
To run API: `dotnet run -p Api` <br>
To just build: `dotnet build` <br>
To run test: `dotnet test` <br>
To run test verbose + logging: `dotnet test -l:"console;verbosity=detailed"`
